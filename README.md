## Princip

> **Tábor je dobrovolnická akce.** Nikdo není placený. Tábor má vyrovnané finance — žádný zisk, žádná tržní hodnota. Motiv pachatele **NENÍ peněžní** — je osobní (pozemek, pomsta, dlouholetá křivda).

## Bezpečnost

- Žádná smrt, žádná viditelná krev.
- Mladší účastníci (7–9) chráněni rámováním („je to hra"), nocí ve stanu se starším vedoucím.
- Pomocník zločinu = **slabý člověk, ne zlosyn** — emocionální moment přiznání.
- Hlavní pachatel = **člověk s motivem, ne karikatura** — důstojné dopadení.
- Závěrečné odhalení patří **všem** (každý dostává dědický list).
